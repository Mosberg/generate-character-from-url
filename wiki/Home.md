# Generate Character From URL v6

Convert any URL, character card, image, PDF, or web page into a character you can import directly into [ai-character-chat](https://perchance.org/ai-character-chat).

**[Try it on Perchance](https://perchance.org/generate-character-from-url)**

---

## What can it do?

- Generate a roleplay-ready character from any URL
- Import character cards from popular sites (Character.AI, chub.ai, JanitorAI, and more)
- Process PDFs, images with embedded metadata, and plain web pages
- Run large URL lists in batch mode
- Export one or many characters as a single importable file

## Key concepts

**Single mode** — paste one URL, click Generate, download the result.

**Batch mode** — paste many URLs, configure retry/timeout/naming, click Start. Results download automatically.

**Combined export** — all successful batch characters packed into one file you import once.

**Global settings** — force every queued character to share the same model, temperature, writing style, etc.
